# BreatheCo.de Desktop Client

This is the application use to deliver the courses.

## Instalation

1. Download the repository
```sh
$ git clone <repo_url>
```
2. Install npm packaes
```sh
$ npm install
```
3. Create your **.env** file and setup the envirnoment variables
```js
// .env

//The wordpress host
WP_HOST=127.0.0.1
```
